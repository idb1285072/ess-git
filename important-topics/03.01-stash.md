# Stash - temporary save uncommitted change

```bash
git stash
git stash push -m "message" # save with message (modern)
git stash save 'message' # save with message
git stash -u # --include-untracked
git stash -a # include --all (untracked + ingnored)
git stash list
git stash show # details
git stash apply
git stash pop # apply and delete
git stash drop
git stash clear
git stash branch fix-login stash@{0} # create branch from stash
```
